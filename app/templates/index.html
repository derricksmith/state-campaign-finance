State Campaign Finance Project

A searchable multi-state campaign finance database

This database includes all campaign contributions filed with state authorities in the supported states. Coverage varies from state-to-state but universally includes elected state officers, state legislators, and state-level political action committees. Local election coverage varies from state to state.
Currently supporting Iowa and Texas, with more states coming soon.
